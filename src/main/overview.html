Gutek Application Overview

Gutek is a Java-based application designed to support intelligent learning through customizable revision algorithms. The application integrates Spring Boot for backend management and JavaFX for the graphical user interface, allowing users to manage language learning, spaced repetition, and memory retention processes.

Main Components

  • gutek: The main package, containing the entry point for the application in the {@code Main} class.
  • gutek.gui: Contains all graphical user interface components, organized in sub-packages for charts, controllers, and custom controls.
  • gutek.services: Provides business logic and service classes, including translation and revision algorithm management.
  • gutek.entities: Contains JPA entities representing key data models, such as cards, decks, and users.
  • gutek.repositories: Spring Data JPA repositories for managing persistence of entities.
  • gutek.utils: Utility classes that provide common functionality, including validation, alert management, and FXML loading.
  • gutek.config: Configuration classes for setting up Spring beans and application properties.

Features

  • Intelligent revision algorithms with support for bidirectional learning
  • Customizable and extensible framework for adding new algorithms and views
  • Automated database management with Spring Data JPA
  • Statistics and analytics to track learning progress

Getting Started

To run the application, start by initializing the Spring Boot context in the {@code Main} class. This will set up the JavaFX interface and display the main stage for interacting with learning tools and settings.
